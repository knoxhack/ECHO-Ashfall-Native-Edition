# ECHO Official Packs

Official packs are curated ECHO profiles. Ashfall is Official ECHO Pack #1 and
demonstrates the full ECHO stack without making Ashfall a dependency of
reusable modules.

Each official pack declares metadata in this folder and may provide a profile
file with module selection, configs, datapacks, resource packs, ThemeCore theme,
Terminal layout, Index content, MissionCore missions, HoloMap layers,
SoundCore rules, WeatherCore events, WorldCore regions, and Lens scan defaults.
