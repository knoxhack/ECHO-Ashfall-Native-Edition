# ECHO Ashfall Native Edition

Ashfall pack/runtime distribution for the ECHO native platform. It consumes `.echo-addon` module artifacts.

## Purpose

Ashfall pack/runtime distribution for the ECHO native platform. It consumes `.echo-addon` module artifacts.

## What Lives Here

Native edition pack manifests, install/update docs, compatibility notes, release notes, rollback guidance, templates, and native-specific issue templates.

## Release And Update Role

Owns Native Edition pack releases and pins `.echo-addon` module requirements resolved from `knoxhack/ECHO-Modules`.

## Public Or Private

Public is recommended for player install/update metadata once native releases are intended for external testers. Keep experimental private branches separate if needed.

## Build And Dev Commands

Run commands from the repository root.

- `No build command is currently tracked in this repo.`

## Artifact Ownership

Pack manifests, pack archives, per-file assets, and native edition release notes belong here. Shared module artifacts stay in `ECHO-Modules`.
