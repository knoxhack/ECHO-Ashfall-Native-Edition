# Pack #2 MVP Scope

The MVP is intentionally small. It should prove that Foundry Protocol works as a second official ECHO experience without overbuilding.

## MVP Must Include

- First 30 minutes: one guided factory commissioning route.
- First mission chain: restore power, route processor, inspect depot.
- First Index entries: overview, grid commissioning, processor route, depot requests, runtime budgets.
- First Terminal layout: facility status, power, throughput, logistics, next task.
- First HoloMap layer: opening factory floor.
- First Lens scan: dormant substation or blocked processor.
- First unique mechanic: factory commissioning checklist using existing modules and pack-specific config/data.
- First docs: support, known issues, module plan, progression notes.
- First theme draft: `pack2_theme` selection with Foundry palette notes.

## Out Of Scope

- Multi-floor campaign.
- New addon creation.
- New public APIs unless separately justified.
- Final screenshots.
- Public launcher listing.
- Full balance pass.
