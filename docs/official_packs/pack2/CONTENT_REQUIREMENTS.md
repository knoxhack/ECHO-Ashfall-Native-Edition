# Pack #2 Content Requirements

Pack #2 is not fully implemented in 1.0.0. The first MVP loop is now playable, and these are requirements before any public visibility.

## Required Before Public Visibility

- A tested first 30 minutes mission chain.
- At least five Index entries for facility operation.
- A Terminal dashboard route based on live MissionCore state.
- One HoloMap waypoint objective plus metadata layers for the opening facility.
- One Lens scan metadata profile plus live Deep Scan objective progress.
- One starter factory mechanic loop using existing PowerGrid content.
- ThemeCore token draft using `pack2_theme`.
- Known issues, support path, release notes, and screenshots.

## Pack-Specific Content

- Facility names, operator copy, production orders, HoloMap markers, mission text, and route records.
- Dedicated launcher identity, icon, banner, card, and showcase images.
- Starter datapack ids under `echo_template_pack2_draft` or a future Pack #2 namespace.

## Reusable Module Candidates

- Generic Terminal data providers for factory status.
- Generic Index categories for machine troubleshooting.
- Generic HoloMap layer type for facility floors.
- Generic Lens scan summary for power/logistics blockers.

Reusable candidates require separate module tickets and must not be smuggled into the Pack #2 profile.
