# Official ECHO Pack #2 Release Candidate Known Issues

Pack #2 is a locked release-candidate prototype for ECHO 1.0.0. It remains approved for controlled creator preview, but it is not public beta, not content-complete, and only covers the first 30-60 minutes of the Foundry Protocol prototype.

## Known Preview Gaps

- Dedicated Pack #2 logo, icon, banner, launcher card, showcase, infographic, and live screenshots are not final yet.
- Current preview assets are placeholders and must not be used as public marketing art.
- The profile contains starter prototype content only; no final campaign, recipes, loot, structures, or worldgen content is implemented.
- The first 30 minutes are validation content, not final public balance.
- Terminal, HoloMap, Lens, SoundCore, TutorialCore, ThemeCore, and WorldCore Pack #2 files are data/profile integration stubs around existing live module systems.
- Optional module warnings are expected when HoloMap, Lens, SoundCore, or other preview modules are absent.
- RuntimeGuard references are advisory only; no Pack-specific runtime budget is required yet.
- Launcher public visibility, storefront readiness, and public release approval remain disabled.
- Install/play buttons must remain disabled or preview-only.
- No new major features should be added after the 1.0.0 lock; bugfixes only.

## Release Boundary

These gaps are expected for the 1.0.0 release-candidate prototype. They are the reason Pack #2 stays out of the 1.0.0 public showcase and should not be treated as public-beta release notes.

Pack #2 should not appear as a public official pack until final art, screenshots, gameplay content, balance, and release approval are complete.
