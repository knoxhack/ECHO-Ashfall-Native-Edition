# Pack #2 Art Prompts

Use these prompts when dedicated Pack #2 art is approved. The style should feel like a clean, operational factory control system: graphite metal, cyan instrumentation, amber caution accents, readable silhouettes, practical machinery, route overlays, and restored industrial spaces. Avoid Ashfall's ruined survival desert language, dust-storm mood, burnt-orange wasteland palette, and collapse-focused imagery.

## 1:1 Logo

Create a 1:1 logo for "ECHO: Foundry Protocol": compact industrial emblem, power-node circle integrated with a conveyor arrow and small diagnostic tick marks, graphite steel, cyan instrumentation glow, amber caution accent, clean vector readability, no dust storms, no survival wasteland, no horror mood, no watermark.

## Launcher Card Icon

Create a readable launcher icon for "ECHO: Foundry Protocol": simplified factory control console, small power conduit loop, machine silhouette, cyan and amber operational lights, strong shape at small size, distinct from Ashfall, no text baked into the icon, no watermark.

## Banner

Wide cinematic banner of a dormant ECHO factory coming online: visible power conduits, battery banks, processor bay, logistics depot board, clean route lighting, cyan dashboard glow, amber caution strips, dark graphite metal, hopeful systems-restored mood, no dust storm, no desert, no ruined survival camp.

## First Screenshot

In-game screenshot target: player standing at the Foundry Terminal with starter PowerGrid parts visible nearby, first mission accepted, internal preview route obvious, no final public claims, readable dashboard composition.

## Terminal Dashboard Screenshot

In-game screenshot target: Foundry Protocol Terminal dashboard showing internal preview badge, next objective, What Now panel, module status, mission panel, archive docs, map objective, scan status, and warning state.

## HoloMap Screenshot

In-game screenshot target: HoloMap factory layer with next-objective marker, processor route, depot route, risk labels, and compact themed icons. Keep the factory plan readable and avoid map clutter.

## Index Page Screenshot

In-game screenshot target: Foundry Protocol Index page open to the preview brief or troubleshooting entry, with related links visible and clean cyan/amber theme tokens.

## Feature Infographic

Create a clean feature infographic for internal review: Terminal -> MissionCore -> Lens -> Index -> HoloMap -> SoundCore -> ThemeCore flow, factory-floor framing, route lines, compact labels, no public release badge, no Ashfall motifs, no watermark.
