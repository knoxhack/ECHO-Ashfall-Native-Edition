# Ashfall Official Pack Changelog 1.5.5

Ashfall is locked as Official ECHO Pack #1 for the final 1.5.x release candidate.

## Changed

- Retargeted Ashfall official metadata, profile, template, example pack, launcher card, and release docs to 1.5.5.
- Locked the existing showcase route across Terminal, Index, Lens, HoloMap, MissionCore, TutorialCore, WeatherCore, SoundCore, WorldCore, RelicTech, PowerGrid, Armory, Logistics, Convoy, and SignalOS bridge data.

## Fixed

- Corrected the Ashfall showcase tutorial hint priority from text to numeric data.

## Isolation

- Ashfall-specific starter data remains scoped to the Ashfall official profile/template/example pack or `echoashfallprotocol` data namespace.
- Generic templates and reusable modules remain Ashfall-optional.
