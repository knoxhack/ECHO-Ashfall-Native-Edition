# Ashfall 1.7.6 Changelog

Ashfall 1.7.6 is the current release-candidate metadata line for Official ECHO Pack #1.

## Changed

- Made the required module closure explicit for the Native, NeoForge, and Standalone pack lanes.
- Retained the Ashfall showcase route while release validation focuses on packaging, Native Loader gates, and real gameplay evidence.
- Kept release blockers visible in `KNOWN_ISSUES.md` until real client, server, export, soak, and balance evidence is collected.

## Release Gate

This changelog does not mark Ashfall public-release-ready by itself. Public release still requires playable artifacts, passing Native Platform gates, complete module artifacts, real gameplay QA, and updated player-facing screenshots.
