# Ashfall - Official ECHO Pack #1

Ashfall is the first official ECHO modpack: a ruined-world sci-fi survival experience built to showcase the ECHO module stack. ECHO modules remain reusable outside Ashfall.

## Status

Beta packaging candidate for ECHO 1.5.4.

## 1.5.4 Showcase Flow

Ashfall now has a clear first-play showcase route:

1. Start at the armored drop pod.
2. Open Terminal and read the Ashfall field dashboard.
3. Read the Ashfall Showcase Flow Index entry.
4. Scan drop-pod wreckage with Lens.
5. Pin the first HoloMap field objective.
6. Accept the Ashfall Showcase Opening MissionCore mission.
7. Respond to the first WeatherCore toxic-front warning.
8. Recover the first RelicTech field note.
9. Repair the pod PowerGrid spine.
10. Install the first Armory upgrade.
11. Follow the first faction/story signal.

Ashfall-specific content stays in `metadata/official_packs/ashfall`, `metadata/templates/ashfall_official`, `examples/packs/full_ashfall`, and `addons/echoashfallprotocol/src/main/resources/data/echoashfallprotocol`. Generic templates and reusable modules must remain Ashfall-optional.
