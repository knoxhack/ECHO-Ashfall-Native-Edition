# Ashfall Support

Use this support path for Ashfall Native Edition release candidates and public test builds.

## Before Reporting

- Confirm the installed pack version and channel in ECHO Launcher.
- Run repair or reinstall from the Launcher before filing install corruption reports.
- Export a support bundle when available.
- Include whether the issue was seen in Native, NeoForge, or Standalone.

## Required Details

- Ashfall version and release tag.
- ECHO Launcher version.
- Java version.
- Operating system.
- Install path.
- Latest client or server log.
- Steps to reproduce.

## Release Blockers

Known release blockers stay in `docs/official_packs/ashfall/KNOWN_ISSUES.md` until validated and cleared.
